### 根因 (Root Cause)

- 任务背景: JC 要 Codex 桌面端 thread sidebar 显示全部 434 条历史 (而不是首屏 50 条)
- 没有 native config / env / CLI flag 控制 page count, 唯一可改字段是 webview JS 里硬编码常量
- Codex 写脚本 `patch_codex_sidebar_full_history.sh`: `npx asar extract` → perl 替换 → `npx asar pack` → cp → `codesign --force --deep --sign -` 重签
- **第二次 patch 时**, asar pack 重打包过程 + codesign --deep 重签 nested code (Squirrel framework / Codex Helper.app) 链条破坏 binary integrity (二进制完整性)
- 启动期 V8 bootstrapper (引导器) 加载 patched JS → 递归打满栈 → SIGTRAP
- 跨重启不可恢复 — JC 每次"重新打开"都加载同一坏 asar, 触发同样崩溃

### Sharp Edge #27 (3 条铁律)

#### 铁律 1: agent self-modify (自我修改) runtime 必 smoke test (启动验证) 5s 才覆盖正式位置

```
改完 app.asar / 二进制 / runtime config → cp 到临时位置 (/tmp/test-*.app)
临时替换 → open -a → sleep 5
检查 ~/Library/Logs/DiagnosticReports/Codex-*.ips 没有新 crash report (崩溃报告)
没崩才 cp 到正式位置 /Applications/Codex.app/Contents/Resources/app.asar
```

ROI: 5s smoke test (启动验证) 投入换 1 小时救火 + 防止跨重启不可恢复, 极高 ROI

#### 铁律 2: patch 必 idempotent (幂等) + 不叠加

- 每次 patch 必须 from **原版** (May 23 干净 backup SHA `cdc98477...`) 重新 patch
- 不能在 patched 版本上再 patch (今晚就是叠加 patch 把 asar 改烂)
- 脚本里加 prerequisite check: `if [ "$(shasum -a 256 $ASAR | awk '{print $1}')" != "$KNOWN_ORIGINAL_SHA" ]; then echo "non-original asar, refuse"; exit; fi`
- 或先 restore 到原版再 patch

#### 铁律 3: fix path (救命草路径) 普通用户 + agent 都要知道

- backup 位置: `~/.codex/backups/codex-app-asar/app.asar.<TIMESTAMP>.bak`
- 找**最早 mtime** 的那份 = 原版 (后续 .bak 都是 patched 中间版本)
- 还原: `cp -p <clean-backup> /Applications/Codex.app/Contents/Resources/app.asar`
- 验证: 启动后没新 crash report + 主进程 PID alive

### 二连 vs 三连决定

**Opus-CSO 拍板: 二连 (changelog + push GitHub), 不升宪法**

理由 4 条:

1. **N=1 实证**, 公司"≥2 升 SOP / ≥3 升宪法"原则未达三连门槛
2. **跨 agent 边界还没验证泛化性** — Codex 桌面端踩了, 但 Hermes 二进制 / Claude Code / Codex CLI / 各 worker agent 三种 case 还没碰过, 真升宪法应该等多 case 抽象提炼边界
3. **Codex CTO 在 019e6092 已自发展示 smoke test 模式** — 没显式宪法铁律, 仅凭今晚 sharp edge memory + prompt inline 强调 + 自身谨慎判断, 已经做对了。链路有效不必升宪法
4. **memory 文件里已写升宪法触发条件**: 任意 worker / C-level 二次踩坑 self-modify runtime → Opus-CSO 立刻升宪法 — 兜底有了
